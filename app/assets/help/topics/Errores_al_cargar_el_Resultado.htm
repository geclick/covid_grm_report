Errores al cargar el Resultado

Ø Archivo mayor de 1 Mb. El tamaño máximo del archivo que se puede subir al sistema es 1 Mb.

 

 

Solución.

1. Abrir el archivo con la opción Abrir y Reparar del excel. Vea Data Higiene .

2. Cargar nuevamente el archivo.

 

 

Ø Formato no reconocido. El formato que debe tener el archivo es xlsx . 

 

 

 

Solución. 

1. Abrir el archivo con excel  

2. Guardar  el archivo con formato .xlsx

3. Cargar nuevamente el archivo.

Ø Laboratorio incorrecto. El excel  cargado no corresponde con el laboratorio procesado

Solución. 

1. Actualizar el Laboratorio (Procesado)

2. Cargar nuevamente el archivo.

Error500. En este caso se debe contactar al administrador del sistema.covidgrm@infomed.sld.cu
