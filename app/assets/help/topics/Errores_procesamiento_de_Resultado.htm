Errores procesamiento de Resultado

Ø  Error 400. La muestra no existe. Muestra el siguiente mensaje: "La muestra no está registrada en la base de datos, verifique!!!!!"

 

Solución :

1. Exportar con el formato de la data de Higiene el/los registros con errores. Vea (3) de Error en procesamiento Resultado .

2. Realizar los cambios necesarios en el excel

3. Cargar el fichero de muestras. Vea (6) de Cargar Muestra

4. Actualizar la carga del resultado. Vea (2) de Cargar Resultados

5. Eliminar el mensaje de error.

 

Ø  Error 400. La muestra es positiva. Muestra el siguiente mensaje: "La muestra ya fue diagnosticada como positiva."

 

 

Solución:

1. Buscar en (1) los detalles del resultado para precisar datos como el código de la muestra, el nombre y los apellidos del paciente, fecha del resultado, etc, 

2. Precisar en la Base de datos de Resultados. Vea Base de Datos de Resultados  y determinar:

    Si la muestra ya existe y el resultado es POSITIVO:

             1. NO se realiza ningún cambio en el sistema

             2. Eliminar el mensaje de error.

    Si es un resultado de un EVOLUTIVO:

1. Exportar con el formato de la data de Higiene el/los registros con errores. Vea (3)  de Error en procesamiento Resultado .            

2. Actualizar el excel con el nuevo código de muestra y cargar la muestra. Vea (6) de Cargar Muestra

3. Actualizar la carga del resultado. Vea (2) de Cargar Resultados         

 3. Eliminar el mensaje de error.

 

Ø FIELD_RENAMER_00. Faltan columnas en el excel

 

Solución

1. Descargar el excel subido. Vea  (4) de Cargar Resultados
2. Corregir el formato según el laboratorio que procesó las muestras. Vea   Formato para Acopio de Datos.
3. Actualizar la carga del resultado. Vea (2) de Cargar Resultados
4. Eliminar el mensaje de error.
