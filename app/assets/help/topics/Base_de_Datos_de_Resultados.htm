Base de Datos de Resultados

La opción  Base de Datos de resultados muestra el listado de pacientes que exiten en PCR_REPORT.

Desde esta vista, se puede:

  1. Visualizar el total de pacientes
  2. Aplicar un Filtro básico para acotar el listado de pacientes.
  3. Aplicar un  Filtro Avanzado para acotar el listado de pacientes
  4. Desplazar por el listado de pacientes mediante la paginación o mostrando todos los resultados en la misma página.
  5. Exportar el listado en un formato de interés.
  6. Añadir un Resultado nuevo
  7. Buscar  por determinado valor.

 

En la columna Resultado, el color ROJO resalta el valor POSITIVO, lo que facilita la ubicación visual al usuario de este tipo de resultado.
