Roles y Permisos en el sistema

El acceso al sistema está delimitado por perfiles o roles, que garantizan la seguridad de la información y el acceso a la misma según la responsabilidad de cada actor en el negocio.

 

Están definidos 3 roles:

 

1.    Administrador

2.    Capturador

3.    Centro

 

Administrador: es el máximo nivel en la jerarquía y tiene acceso a:

Ø  Todas las funcionalidades que implementan el negocio, es decir, accede a las funcionalidades del Capturador y del Centro de Aislamiento.

Ø  Además a la configuración de aspectos técnicos  como la gestión de nomencladores, usuarios, permisos, etc.

 

 

 

 

Capturador: El capturador de información, por lo general es quien verifica que cada excel recibido de los  laboratorios de Epidemiología del país cumplen con el formato establecido para ser procesados satisfactoriamente en el sistema. Además es el responsable de implementar el negocio y tiene acceso a:

Ø  Cargar muestras

Ø   Filtrar muestras.

Ø  Cargar resultados

Ø   Filtrar listado de resultados

Ø  Listar la base de datos de resultados .

Ø  Filtrar listado de la base de datos de resultados aplicando Filtros Básicos y Avanzados .

Ø  Buscar resultados  específicos en la base de datos de resultados.

 

 

 

 

Centro: El especialista de cada centro de aislamiento solo tiene acceso a:

Ø  Listar la base de datos de resultados .

Ø  Filtrar listado de la base de datos de resultados aplicando Filtros Básicos y Avanzados .

Ø  Buscar resultados  específicos en la base de datos de resultados.
