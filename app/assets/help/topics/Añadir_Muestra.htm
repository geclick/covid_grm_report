Añadir Muestra

La opción Añadir Muestra permite procesar un nuevo archivo en el sistema.

Errores más comunes al añadir el archivo para su procesamiento:

  1. Archivo mayor de 1Mb
  2. Formato no reconocido.

Para saber cómo proceder ante cada error, vea Errores al cargar la Muestra
